# Adstera Marketing Site (Next.js)

This repository now uses **Next.js (App Router + TypeScript)** to serve the Adstera marketing website.

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Production build

```bash
npm run build
npm run start
```

## Deployment and custom domain

Deploy on Railway or Vercel using the default Next.js settings (`npm run build` + `npm run start` for Railway, or framework auto-detect on Vercel).

For custom domain setup (e.g. `app.adstera.ink`):

1. Add the domain in your hosting provider dashboard (Railway/Vercel project settings).
2. Point DNS records from your DNS provider to the target generated by the host.
3. Verify domain and enable HTTPS in the hosting dashboard.
